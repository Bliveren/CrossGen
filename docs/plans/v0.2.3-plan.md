## 版本目标

v0.2.3 是一个**小版本补丁**，解决两个阻塞后续发布的问题：

1. **PR #118 — 禁用全部 OpenAI 流式请求**：聚合站对 `/images/generations` 也拒绝 SSE，彻底关闭 streaming 以兼容所有提供商。
2. **解除 `pnpm build` 阻塞**：后端已将 `AppSnapshot.config` 改为 `providers[] + activeProviderId`（阶段 0 WIP），但 `App.tsx` / 测试尚未跟进，导致 74 个 typecheck 错误，`pnpm package:mac:signed` 无法运行。

完成后可立即执行签名打包并发布 v0.2.3。

## 新增功能

### 图片预览区右键菜单

预览区（`result-canvas`）和模态预览的图片上，右键弹出上下文菜单，包含：

- **保存图片**（`bridge.downloadAsset`）
- **打开文件夹**（`bridge.openAssetFolder`）
- **复制提示词**（复制当前 job 的 prompt）

实现方式：自定义 React 状态驱动的 DOM 菜单（不依赖 Electron menu IPC），`onContextMenu` 触发，点击外部/Escape 关闭。
