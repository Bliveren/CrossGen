## OpenAI Streaming Compatibility

- Direct `https://api.openai.com/v1` OpenAI generation supports `stream: true` and `partial_images` by default when `n === 1`.
- Custom OpenAI-compatible gateways and aggregators default streaming off. Users can explicitly enable streaming per API configuration after verifying that the endpoint supports SSE image events.
- `n > 1` continues to downgrade to non-stream JSON requests.
- Edit and inpaint continue to use non-stream requests by default for compatibility.
- If a stream generation request is rejected as incompatible, the OpenAI adapter retries the generation as non-stream JSON and keeps the non-stream path available.

## Residual Risks
