## Compatibility Note

The legacy `pnpm verify:real-api` direct `/images/edits` path was also tried
against AIHub. Text-to-image returned an output, but single-image edit returned
empty `data`, matching the provider compatibility issue seen during app testing.
The release gate therefore validates the product-supported AIHub
`/chat/completions` streaming path instead of treating the incompatible direct
`/images/edits` path as release-blocking.
