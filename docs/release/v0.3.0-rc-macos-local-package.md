## Commands

```bash
pnpm build
pnpm verify:mock-api
pnpm verify:mock-gemini-api
pnpm verify:mock-model-discovery
pnpm verify:release-evidence
pnpm verify:signing-ready
pnpm package:mac
pnpm verify:release:mac
```

`pnpm verify:signing-ready` intentionally failed because the release shell did not define `CSC_NAME`, `APPLE_ID`, `APPLE_APP_SPECIFIC_PASSWORD`, or `APPLE_TEAM_ID`. It did find one valid local code-signing identity and confirmed that `package:mac:signed` now provides a Developer ID identity override path when the required environment is configured.
